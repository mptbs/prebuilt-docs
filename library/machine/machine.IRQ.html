Class IRQ – Handle interrupt requests¶

Constructors¶

An IRQ object cannot be explicitly created, but is created and returned when a peripherals interrupts are enabled via its irq-function such as Pin.irq(...).

Class methods¶

IRQ.init()¶

Reinitializes the interrupt. The interrupt will be automatically enabled.

IRQ.__call__()¶

Manually call the interrupt handler.

IRQ.enable()¶

Enable the interrupt.

IRQ.disable()¶

Disable the interrupt.

IRQ.handler([handler])¶

Set or get handler function

IRQ.flags()¶

Get the triggers that caused the current interrupt. Only returns useful values when called inside the interrupt handler.
