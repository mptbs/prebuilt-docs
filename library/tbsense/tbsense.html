tbsense — Functions related to the board¶

Module functions¶

tbsense.reset_module()¶

Turn off power to all sensors on the board, including the RGB LEDs.

tbsense.reinit_peripherals()¶

Reinitialize I2C0 and SPI0 (USART0) to a configuration suitable for using the sensors on the board.

Time related functions¶

tbsense.delay(ms)¶

Delay for the given number of milliseconds.

tbsense.deepsleep(s)¶

Deepsleep for the given number of seconds. Enters EM2.

tbsense.ms()¶

Return the number of milliseconds since the last board reset.

The returned result is a micropython smallint (31-bit signed number), so after 2^30 milliseconds (12.4 days) this number will become negative.

tbsense.ms_elapsed(start)¶

Return the number of milliseconds since `start.

This function takes care of counter wrap and can be used to measure periods of up to 12.4 days.

Example:

import tbsense

start = tbsense.ms()
while tbsense.ms_elapsed(start) < 1000:
    #Do something!
